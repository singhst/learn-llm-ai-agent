# learn-llm-ai-agent

This repository is a small learning collection for working with Microsoft Agent Framework (MAF), LangChain, and agent-style notebook examples by running LLM with Ollama. The Python code and notebooks are meant for study and experimentation only.

# Important note:  All code in this repo is for learning and experimentation. No code is copied from production code from real world systems. The examples are assembled from public sources, internet references, or generated by AI tools.


## .py and notebook briefly explain

- `test-react_pattern-langchain.py`
  - LangChain ReAct agent demo using `ChatOllama` and two small tools: stock price lookup and profit calculation.

- `test-react_pattern-maf.py`
  - Minimal MAF-style example that runs a local Ollama agent with a finance prompt and streamed output.

- `test-agent_loop-react_pattern-maf.py`
  - MAF example that runs a weather assistant agent with simple tools for location, date, and historical weather.

- `test-agent_loop-react_pattern-maf-show-log.py`
  - Demonstrates streaming monitoring of agent tool calls and observations for debugging.

- `test-agent_loop-react_pattern-maf-show-log-opemtelemetry.py`
  - Shows how to add OpenTelemetry spans and console export for agent executions.

- `test-agent_loop-langchain-tool_use-check-location-date-weather.ipynb`
  - Notebook building a ChatOllama + `langgraph` flow with location/date/weather tools.

- `test-agent_loop-react_pattern-langchain-tool_use-check-location-date-weather.ipynb`
  - ReAct-style LangChain notebook demonstrating tool decision and action structuring.

- `test-agent_loop-react_pattern-maf.ipynb`
  - Notebook wrapper that runs the MAF-related Python example scripts in a virtualenv.

- `test-langchain-ollama.ipynb`
  - Examples of `langchain_ollama` usage, prompt templates, and graph-based agent flows.

- `test-ollama.ipynb`
  - Direct `ollama` Python API demos: chat, streaming, generate, and LangChain wrapper examples.
